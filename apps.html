首选 · 一行命令

软件仓库

每个发行版都自带官方仓库，几万个软件经过审核、自动更新。安装就是一句话的事：

# Debian / Ubuntu / Mint
sudo apt install gimp
# Fedora
sudo dnf install gimp
# Arch（有咲同学钦定）
sudo pacman -S gimp
鼠标党友好

应用商店

GNOME「软件」、KDE「发现」都是系统自带的图形化商店：搜索、点安装、完事。deepin 商店和星火应用商店对国内软件收录更全，中文描述看着也亲切。

商店背后连的还是软件仓库与 Flatpak，和命令行装的是同一批货，谁也不比谁高贵。

跨发行版 · 版本新

Flatpak / Flathub

软件仓库里的版本太旧？Flathub 上几千款应用跟随上游更新，任何发行版都能装：

flatpak install flathub md.obsidian.Obsidian

沙盒隔离运行，卸载不留痕，适合尝鲜新版本。

兜底 · 三板斧

官网包 / AppImage / AUR

  • 官网安装包：WPS、QQ 等提供 .deb / .rpm，下载双击即装
  • AppImage：单文件应用，赋予执行权限双击就跑，免安装
  • AUR（Arch 限定）：十万个社区维护的软件包，yay -S 搜什么有什么——有咲同学的原话：「这才是 Arch 的真正本体……别、别到处乱说。」

底牌一 · 兼容层

Wine

让 Windows 软件直接跑在 Linux 上，LTspice、部分老旧专业课软件都靠它续命。配 Bottles 图形化管理，基本不用碰命令行。Proton 就是它的游戏特化版。

底牌二 · 虚拟机

KVM 虚拟机

virt-manager 把整个 Windows 塞进一个窗口，性能接近物理机。微信开发者工具、ANSYS 学生版这类「实在没辙」的软件，开虚拟机十分钟收工。

底牌三 · 最后的倔强

双系统 / 网页版 / 云

反作弊网游、剪映这类死硬派：双系统留一个 Windows 分区兜底，或者用云游戏、网页版曲线救国。哪些软件属于这一类，避坑指南里写得很诚实。
