🐣 入门概念

发行版Distro

Linux 的「品牌套餐」：同一个内核，配上不同的软件、桌面和理念打包出售。Ubuntu、Arch、Debian 都是发行版——就像同样的发动机，装进了不同品牌的车。

内核Kernel

系统真正的核心，负责跟硬件对话。你平时说的「Linux」严格来说只是它，平时用的完整系统其实是「内核 + 一堆软件」的组合。

桌面环境DE

你看到的桌面、任务栏、设置中心的总和。GNOME 像 macOS，KDE 功能多到像瑞士军刀，Xfce 轻快适合老电脑。同一个发行版可以换着用。

包管理器Package Manager

Linux 的应用商店 + 自动更新二合一，不过是命令行版的。一行命令装软件，还自动处理依赖。各家的叫法：Ubuntu 用 apt，Arch 用 pacman，Fedora 用 dnf。

软件仓库 / 源Repo

发行版官方维护的软件大超市，包管理器从这里进货。说它「官方」是因为里面的软件都经过审核，比在网页上乱下安装包安全得多。

镜像站Mirror

软件仓库在国内的「分店」。默认源在国外下载慢，换成清华、中科大等镜像站，装软件速度起飞。换源是装完系统的第一课。

滚动更新Rolling Release

没有「大版本」，系统天天小步快跑，软件永远最新。优点是新鲜，代价是偶尔会「滚挂」——更新完出毛病。Arch 就是滚动派掌门。

LTS长期支持版

追求稳定的版本策略：一个版本官方维护 3～5 年，只修 bug 不追新。求稳的同学认准 LTS 三个字母，比如 Ubuntu 24.04 LTS。

Live USB试用 U 盘

把系统装到 U 盘里开机就跑，不碰硬盘。买电脑、装系统前的「试驾」神器：先在 Live 模式里试试 Wi-Fi、触板好不好使，再决定装不装。

基于 / 血统Based on

「Mint 基于 Ubuntu，Ubuntu 基于 Debian」——发行版也有族谱。基于谁，就继承谁的软件仓库和脾气，所以查教程时血统比名字更重要。

⌨️ 命令行黑话

终端 / ShellTerminal / Bash

终端是那个黑窗口，Shell 是里面听你指挥的「管家」（Bash、Zsh 都是 Shell 的品种）。日常混用没问题，别在考试里较真。

sudo

「以管理员身份执行」。相当于 Windows 的右键管理员运行。经典梗：说不通的人，在前面加个 sudo 就通了——sudo 帮我带饭。

root

系统里的超级管理员，权力大无边，也能一句话把系统扬了。平时用普通用户 + sudo 就好，别图省事常驻 root。

家目录~

你的个人文件夹，相当于 Windows 的「用户目录」，用 ~ 表示。你的文档、配置、黑历史都在这。备份先备它。

管道|

把前一个命令的输出塞给后一个命令当输入，像流水线接力。例：ps aux | grep qq ——先列出全部进程，再筛出带 qq 的行。

重定向> 和 >>

把命令的输出写进文件：> 是覆盖写，>> 是追加写。例：echo "我爱 Linux" > 日记.txt。

守护进程Daemon

在后台默默干活的程序，比如 Web 服务、打印服务。名字听着吓人，其实是因为英文 daemon 原意是「守护精灵」，不是恶魔。

chmod 777

给文件「所有人都能读写执行」的权限。社区著名反面教材：出问题就 777 是莽夫行为，正确做法是搞清楚谁需要什么权限。

TTY

纯命令行的全屏终端。图形界面卡死时按 Ctrl+Alt+F3 能进 TTY 救场，按 Ctrl+Alt+F2（或 F1）切回图形界面。记住它，关键时刻救命。

🏘️ 社区黑话

AURArch 用户仓库

Arch 用户自发上传的软件仓库，十万软件包什么都有。是 Arch 最大的魅力，也要求你留个心眼——毕竟是网友上传的，装前瞄一眼评论。

PPA

Ubuntu 的第三方软件源，用来装官方仓库没有（或太旧）的软件。相当于给应用商店加了个「代购渠道」。

RTFM

「Read The Fine Manual」——去读手册！社区回答伸手党的经典回复。礼貌版说法是「这个在 Arch Wiki 上有哦」。提问前先搜索，是社区第一课。

Distro Hopping

发行版跳跃症：不停地换系统尝鲜，Mint 用两周换 Fedora，再换 Arch……老玩家的必经之路。放心，大部分人会最终在某一站安家。

滚挂

滚动更新翻车的简称：一更新，系统出毛病了。应对：更新前看一眼公告，装好快照工具，滚挂了也能十秒回滚。

日用

把 Linux 当主力系统天天用。「这个发行版能不能日用」=「靠不靠谱，能不能托付四年」。

炼丹

训练 AI 模型的戏称：参数是药材，显卡是丹炉，loss 不降就是丹炸了。Linux + NVIDIA 是炼丹界标准道观。

Ricing

桌面美化装修：换主题、调毛玻璃、配状态栏，把桌面打扮得比 macOS 还好看。源自汽车圈的「爆改」，是 Linux 玩家的重要快乐源泉。

「BTW, I use Arch」

「顺便一提，我用 Arch」——调侃 Arch 用户逢人必提的梗。你在本站首页看到的有咲同学就是典型患者。

FOSS / 自由软件

Free and Open Source Software。自由软件的「自由」是自由的自由，不是免费——虽然通常也免费。GPL 协议还带「传染性」：用了我的代码，你的也得开源。

上游 / 下游Upstream

软件的原始作者是上游，打包改造的发行版是下游。「报 bug 要报给上游」= 找亲妈告状，别找错人。

企鹅 / Tux

Linux 吉祥物，一只憨憨的企鹅。自称「企鹅人」的，就是 Linux 用户的身份证。来源：Linus 本人被企鹅咬过（是真的）。

🔧 装机与硬件

UEFI / BIOS

开机最先跑的固件程序，负责把系统叫醒。UEFI 是 BIOS 的现代版。装系统、关 Secure Boot 都在它的设置界面里操作（开机猛按 F2/Del/F12）。

Secure Boot安全启动

UEFI 的一项安全检查，有时会拦住 Linux 启动盘。装系统卡在这一步就去 BIOS 里把它关掉——这是新手装机最高频的坎。

分区 / 挂载

把硬盘切成几块用（系统、数据分开），「挂载」就是把某块分区接到目录树的某个位置。双系统就是给两个系统各分一块地盘。

双系统Dual Boot

一台电脑装两个系统，开机时三秒选一个。Windows 当游戏厅，Linux 当书房，成年人不做选择。新手上路最稳妥的姿势。

核显 / 独显

核显长在 CPU 里，省电够用，Linux 下驱动开源免折腾；独显是独立显卡，性能强适合游戏和炼丹，NVIDIA 的驱动是闭源的（但现在一键能装）。

开源 / 闭源驱动

Intel、AMD 显卡驱动是开源的，系统自带；NVIDIA 官方驱动闭源，要单独装。开源替代品 Nouveau 性能一般，打游戏还是用官方驱动。

swap交换分区

硬盘上划出一块当「应急内存」，内存不够时救场，也是休眠功能的存放点。内存 16G 以上给 4～8G 意思一下就行。

快照 / 后悔药Timeshift

给系统拍照片存档，折腾坏了十秒回滚到昨天。和「黑话」没关系，但每个老玩家都会拍着大腿劝你装一个。

词典里还没有这个词——去 GitHub 仓库提个 Issue，我们给它收录进来。
