三分钟换好源

  1. 打开上面任意镜像站，找到「使用帮助 / Help」入口。
  2. 在帮助列表里选中你的发行版与版本号。
  3. 按页面给出的命令或配置，替换系统里的源地址。
  4. 刷新软件缓存，享受起飞速度。

以 Ubuntu 为例

把默认的国外源一键替换为清华源，再刷新缓存：

# Ubuntu 23.10 及更早
sudo sed -i 's|//.*archive.ubuntu.com|//mirrors.tuna.tsinghua.edu.cn|g' /etc/apt/sources.list

# Ubuntu 24.04 起改用新格式，编辑这个文件
sudo nano /etc/apt/sources.list.d/ubuntu.sources

# 最后刷新缓存
sudo apt update

其他发行版同理——Debian、Fedora、Arch 的换源命令，镜像站帮助页里都替你写好了，照抄就行。

小提示：很多高校的校园网对校内和教育网镜像不计流量。换源不仅提速，还可能帮你省下一整个月的流量包——具体政策问你们学校网络中心，或者问下面会提到的校内社团。

Asking

提问的智慧。

在社区里，问题问得好，答案来得快。大佬不是不愿意帮忙，是不愿意帮「在吗」。

提问前，先做这三件事

  1. 把报错信息原样复制去搜——大概率有人踩过同一个坑。
  2. 翻一遍上面的 Arch Wiki 与你的发行版文档。
  3. 读一遍神文《提问的智慧》中译版，十分钟，受益四年。

做完这三步还解决不了？现在可以提问了，而且你已经比 80% 的提问者专业。

提问时，带上这些信息

报错复制文本，别拍屏别截半个屏幕。附上你的环境，大佬一眼就能定位：

# 你是什么系统、什么内核
cat /etc/os-release
uname -r

# 本次开机以来的错误日志
journalctl -b -p err

问题解决后回帖说一句「已解决，方法是……」——下一个搜到这个帖子的人会隔空感谢你。

Loadout

进社区之前，先配好装备。

黑话翻译器

论坛帖子里的发行版、AUR、炼丹、Ricing……每个黑话都有人话版翻译。逛社区之前先翻翻它，帖子立刻从加密通话变成普通话。

命令行小抄卡

查命令去小抄卡，查概念去翻译器，两件装备配齐，社区里 90% 的帖子你都能看懂。

On Campus

校园里，找到组织。

校内开源 / Linux 社团

越来越多高校都有开源社团、Linux 用户组（LUG）或 CTF 战队——QQ 群搜索「你的校名 + Linux / 开源」，大概率有惊喜。加入他们的好处：校园网认证教程、软件课避坑、二手硬件情报，全都是现成的。

镜像站背后的他们

你可能不知道：清华 TUNA、中科大 LUG、上交 SJTUG 这些镜像站，本身就是学生社团在运营。招新不设技术门槛，维护镜像、写文档、办活动都缺人——进去打杂一年，简历和实力一起毕业。
