Cheatsheet

命令行小抄卡。

不用背，收藏这页就行。用的时候搜一下。

📁 文件与目录

ls -lah列出文件（含详情）
cd /路径进入目录
pwd我在哪儿
cp -r 源 目标复制
mv 源 目标移动 / 重命名
rm -ri 目录删除（带确认）
mkdir -p a/b/c递归建目录
find . -name "*.pdf"按名找文件
du -sh *各文件占多大
df -h磁盘还剩多少

📄 查看与编辑

cat 文件一口气看完
less 文件翻页看（q 退出）
head -n 20 文件看前 20 行
tail -f 日志盯着日志实时看
grep -rn "关键词" .全文搜索
wc -l 文件数行数
nano 文件新手友好编辑器

📦 装软件（三家对照）

sudo apt install 包Ubuntu/Debian 安装
sudo dnf install 包Fedora 安装
sudo pacman -S 包Arch 安装
apt search / dnf search搜索
pacman -Ss 关键词Arch 搜索
sudo apt update && sudo apt upgradeUbuntu 更新
sudo pacman -SyuArch 更新
sudo apt remove 包卸载
flatpak install flathub 应用万能备胎方案

🔐 权限

sudo 命令以管理员执行
chmod +x 脚本.sh给执行权限
chmod 644 文件标准文件权限
sudo chown 用户:组 文件改所有者

⚙️ 进程与系统

htop任务管理器（好看版）
ps aux | grep 名找进程
kill PID结束进程
systemctl status 服务查服务状态
sudo systemctl enable --now 服务开机自启并启动
free -h内存用量
uname -a内核信息
fastfetch展示你的系统（仪式感）

🌐 网络

ping baidu.com通不通
ip a看 IP
curl -O 网址下载文件
wget 网址下载文件
ss -tlnp谁在监听端口
ssh 用户@服务器远程登录
scp 文件 用户@服务器:/路径传文件上服务器
rsync -avP 源 目标同步目录（可断点续传）

🗜️ 压缩与解压

tar -czvf 包.tar.gz 目录打包压缩
tar -xzvf 包.tar.gz解压
unzip 文件.zip解压 zip
zip -r 包.zip 目录打成 zip

🌿 Git 基础

git clone 地址克隆仓库
git status现在什么状态
git add .暂存全部改动
git commit -m "说明"提交
git push推送到远程
git pull拉取最新
git log --oneline看提交历史

❓ 忘了就问

man 命令官方说明书
命令 --help快速提示
tldr 命令人话版例子（需安装）

没搜到？去 资源页 翻翻，或者到社区里问问。
