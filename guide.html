0

装前 30 分钟：备份与退路

翻车概率很低，但退路铺好，手才不抖。
最重要的一步

备份，备份，还是备份

  • 论文、照片、代码：传网盘或拷移动硬盘，双份更安心
  • BitLocker / 设备加密：Windows 设置里搜「设备加密」，先关掉；或确认恢复密钥已存进微软账号——否则装完 Windows 可能问你要 48 位密钥
  • 浏览器书签与各类账号密码：顺手导出一份

记住：新手装系统出的事故，九成不致命；但没备份的资料，是十成找不回。

小动作，省大麻烦

三个装前小动作

  • 关掉 Windows「快速启动」：控制面板 → 电源选项 → 选择电源按钮的功能 → 取消勾选。不关的话，Linux 读 Windows 分区会报「磁盘不干净」，双系统时间还会互相漂移
  • 准备一个 8GB+ 的 U 盘：里面数据会被清空，先转移
  • 记下 Wi-Fi 密码：安装和 Live 试用时都要联网

万一装完发现无线网卡不识别（少数 Realtek / 博通型号会这样），别慌：手机数据线插上，开「USB 网络共享」就能先上网装驱动。

1

选一条适合你的路线

按胆量从小到大排列，没有谁更高贵。
零风险 · 尝鲜首选

虚拟机

  • 适合：只想先看看 Linux 长什么样
  • 工具：VMware Workstation（个人使用免费）、VirtualBox（开源免费）
  • 分配建议：内存 4GB+、CPU 2 核+、磁盘 25GB+，体验才不卡
  • 优点：玩坏了直接删掉重来，Windows 毫发无损
  • 注意：需要 CPU 虚拟化（BIOS 里的 VT-x / AMD-V）已开启；跑不动 AI 和大型游戏
只要命令行 · 开发向

WSL2

  • 适合：只需要 Linux 命令行环境的计科同学
  • 安装：管理员 PowerShell 一行 wsl --install，重启即得 Ubuntu
  • 优点：和 VS Code 无缝联动（WSL 扩展），编译、跑服务、学命令行都够用；文件还能和 Windows 互访
  • 注意：它本质上是住在 Windows 里的 Linux——写作业满分，体验完整桌面生态请走另外两条路
真·主力 · 推荐

双系统

  • 适合：想把 Linux 当日常主力的人
  • 做法：Windows 磁盘管理里「压缩卷」腾出 60GB+ 空闲空间，安装器选「与 Windows 共存」
  • 优点：性能满血，开机菜单自选系统，Windows 留着兜底
  • 注意：装前完成第零步的备份；Windows 大更新偶尔要吃掉引导菜单，一根启动盘跑个 Boot Repair 就能救回
路线上手难度性能风险一句话建议
虚拟机★☆☆打八折几乎为零大一第一学期的稳妥开场
WSL2★☆☆接近原生几乎为零计科专业课的最短路径
双系统★★☆满血低（备份后）想当主力，就直接上
单系统★★☆满血无退路老电脑焕新的最佳选择

老电脑想直接单系统？那更好——少一个系统，少一堆事。旧机器装个 Mint 或 Debian，直接焕发第二春。

选一个发行版

选择困难症发作的话，就闭眼选前两个之一。
Ubuntu教程全网最多，搜什么都有答案
Linux Mint最像 Windows，零适应期
Fedora新而稳，笔记本兼容好
Pop!_OSNVIDIA 显卡开箱即用
deepin国产出品，中文体验拉满
Nobara游戏特化，预装兼容层

拿不准？做个一分钟小测试，让九位同学给你发本命发行版。记住一条社区铁律：对新手来说，教程多的发行版，就是最好的发行版。

3

制作启动盘，然后安装

写盘 10 分钟，安装 20 分钟，剩下时间喝奶茶。

1. 下载镜像

去发行版官网下 .iso，教育网用户强烈建议直接去清华、中科大等镜像站下，快还免流量。没想好装哪个？回第二步，或者无脑 Ubuntu / Mint。

2. 写入 U 盘

Windows 上推荐 Ventoy（开源神器）：给 U 盘装一次，以后把 iso 直接拖进去就能启动，还能多个系统共存。balenaEtcher 和 Rufus 也行，图形界面傻瓜式操作。

3. 开机进 U 盘

重启时按启动菜单键选 U 盘，进入 Live 模式——不安装也能先试用。确认下面清单全部正常，再双击桌面上的「安装」。

Live 模式检查清单

  • Wi-Fi 能搜到并连上
  • 声音外放、耳机都出声
  • 亮度调节、触控板手势正常
  • 合盖再打开，睡眠唤醒不黑屏
  • 屏幕分辨率没有奇怪的拉伸

全部通过 = 这台机器和 Linux 八字相合，放心装。某一项不灵也多半有驱动可救，先记下来装完再处理。

安装时的三个关键点

  • Secure Boot：Ubuntu / Fedora 系直接支持，不用关；少数发行版需要进 BIOS 关掉它
  • 分区：双系统选「与 Windows 共存」让安装器自动分；手动分的话 / 给 50GB+，再留一个 512MB–1GB 的 EFI 分区
  • 用户名：全小写、无空格、无中文，未来的你会感谢现在的你

⚠️ 老手们爱说的 dd 命令写盘，写错盘符就是当场火化，新手请用图形工具。

各品牌开机启动键速查

  • 联想：F12（或机身侧面的 Novo 小孔）
  • 戴尔：F12　　惠普：F9（先按 ESC 也行）
  • 华硕：ESC　　小米 / Redmi：F12
  • Mac：开机长按 Option
  • 台式组装机：常用 F12 / F11 / F8，进 BIOS 是 Del

开机第一屏 logo 下方一般也写着按键提示，手快一点就能看到。

4

装完后的第一件事

按顺序做完这四件，体验直接从「能用」变「好用」。
网络优化

① 换国内镜像源

默认软件源在国外，下载慢到怀疑人生。换成国内镜像，校园网内多半还免流量：

  • 清华 TUNA：mirrors.tuna.tsinghua.edu.cn
  • 中科大 USTC：mirrors.ustc.edu.cn
  • 阿里云：mirrors.aliyun.com　网易：mirrors.163.com

各镜像站首页都有「使用帮助」，搜你的发行版，复制粘贴即可。换完顺手跑一次系统更新。

日常体验

② 中文输入法与显卡驱动

  • 输入法：fcitx5 + 中州韵（Rime）或自带拼音，开箱体验不输 Windows；搜狗也有 Linux 版可选
  • NVIDIA 显卡：Ubuntu 系跑一句 sudo ubuntu-drivers autoinstall 完事；也可以在「附加驱动」里点选
  • 高分屏同学：显示设置里把缩放调到 125%–150%，字瞬间不瞎眼
安全感

③ 给系统上一份保险

  • 装 Timeshift 做个系统快照，折腾坏了十秒回滚——成年人的安全感
  • 开启自动安全更新，其余更新想啥时候点都行
  • 重要资料依旧遵守第零步的备份纪律：系统有快照，数据有云盘
生产力

④ 把软件装满

  • 注册 Flathub 仓库：图形软件一键装，省心；多数发行版的软件中心里点一下就行
  • 浏览器、VLC、Obsidian、Zotero、WPS……想要的几乎都有
  • 找不到 Windows 同款软件的替代品？去软件对照表按专业查

第一周，和终端交个朋友。

别怕那个黑窗口，它不吃人。记住这五条，你已经超过 80% 的用户。

ls                        # 看看当前文件夹里有啥
cd 文件夹名               # 走进某个文件夹
sudo apt install 软件名   # 装软件（Debian/Ubuntu/Mint 系）
man 命令名                # 不会用？问它自己
sudo !!                   # 上条命令忘了 sudo？补票专用

更多命令收在命令行小抄卡里，可搜索、可收藏。看到 rm -rf 开头的「教程」请直接关掉网页，那是社区保留整活节目。

国内高校特供问题

校园网认证

网页认证（Portal）：恭喜，浏览器打开就完事，Linux 毫无障碍。

锐捷 / 深澜等客户端认证：有点麻烦但有解法——

  • 先加学校的 Linux 用户群 / 开源社团群，大概率有现成教程和打包好的客户端
  • 第三方开源客户端（如各类 minieap、drcom 实现）可解大部分学校
  • 终极方案：一台几十块的路由器刷 OpenWrt，让它替你认证，全宿舍设备跟着沾光

微信、QQ 能用吗？

能，而且这两年进步很大：

  • 微信：已有原生 Linux 版（NT 架构，官网提供 deb/rpm），日常聊天、文件传输都没问题，小程序等功能在逐步补齐
  • QQ：NT 架构新 QQ 官方支持 Linux，x86 和 ARM 都有，体验相当现代
  • 会议/办公：腾讯会议、飞书、钉钉均有官方 Linux 客户端；WPS 也有功能完整的 Linux 版

简单说：2026 年的今天，通讯办公已经不再是 Linux 的拦路虎了。

丑话说前面

教务系统与考试软件

选课、查成绩这类教务系统绝大多数是网页版，Linux 浏览器直接用；知网、万方查文献也没问题。真正的坑是少数机考练习客户端（比如某些等级考试的模拟软件）只有 Windows 版——解法有三：双系统切回去、学校机房、或借室友电脑。这也是为什么我们推荐双系统：Linux 负责日常，Windows 负责兜底。

新手 FAQ

虚拟机：直接删虚拟机文件，就当没发生过。双系统：从 Windows 磁盘管理删掉 Linux 分区，再用启动盘修复一下引导即可。单系统：重装回 Windows 也就是半小时的事。资料按第零步备份过，系统本身就是耗材——这正是 Linux 最适合练手的原因：折腾成本约等于零。

大概率比你想象的能跑。4GB 内存选轻量桌面（Xubuntu、Linux Mint Xfce 版），8GB 就随便挑；十年前的机器装 Debian 还能再战三年。选购新机或想查具体机型兼容性，去硬件页看评级。

GNOME 和 KDE 的默认颜值早已在线，deepin 更是精致路线。想进一步折腾主题、 dock、状态栏，社区管这叫「Ricing」，成品可以去 r/unixporn 围观（听不懂这个词？黑话翻译器伺候）。

三件套：① Arch Wiki——写得太好，用别的发行版也值得查；② 你发行版的中文论坛与社群；③ 本校开源社团 / 镜像站用户群。提问礼仪：说清发行版和版本、贴上完整报错截图、说明你试过什么。做到这三点，大佬们抢着回答。

安装界面和桌面都有完整中文，日常使用零门槛。报错信息多为英文，但复制下来丢给搜索引擎或 AI，基本一搜一个准。把报错看懂的过程，本身就是英语和计算机双修的隐藏福利。
