Getting Started

从心动到开机，
只要一个下午。

装 Linux 没有传说中那么玄乎。选对路线，一个下午、一个 U 盘、一杯奶茶，搞定。

「别怕！连我都一次装好了，你一定也可以！」
丸山彩 · Linux Mint 用户 · 本页向导

先想清楚一件事：你想怎么用它？下面三条路线，按胆量从小到大排列。

第一步：选一条适合你的路线

零风险 · 尝鲜首选

虚拟机

  • 适合：只想先看看 Linux 长什么样
  • 工具：VMware Workstation（个人免费）、VirtualBox（开源）
  • 优点：玩坏了直接删，Windows 毫发无损
  • 注意：分 4GB 以上内存体验才不卡；跑不动 AI 和大型游戏
只要命令行 · 开发向

WSL2

  • 适合：只需要 Linux 命令行环境的计科同学
  • 安装：管理员 PowerShell 一行 wsl --install，重启即得 Ubuntu
  • 优点：和 VS Code 无缝联动，编译、跑服务、学命令行都够用
  • 注意：默认没有完整桌面；systemd 服务支持有限
真·主力 · 推荐

双系统

  • 适合：想把 Linux 当日常主力的人
  • 做法：Windows 里压缩出 60GB+ 空闲分区，安装器选「与 Windows 共存」
  • 优点：性能满血，保留 Windows 兜底
  • 注意：装前先备份重要文件；Windows 大更新偶尔要吃掉引导，一根 U 盘就能救

老电脑想直接单系统？那更好——少一个系统，少一堆事。旧机器装个 Mint 或 Debian，直接焕发第二春。

第二步：制作启动盘（10 分钟）

1. 下载镜像

去发行版官网下 .iso。没想好装哪个？先去做个小测试，或者无脑 Ubuntu / Mint。

2. 写入 U 盘

Windows 上推荐 Ventoy（开源神器）：装一次，以后把 iso 直接拖进 U 盘就能启动，还能多系统共存。balenaEtcher 也行，图形界面傻瓜式。

3. 开机进 U 盘

重启按 F12 / F9 / ESC（因品牌而异）选 U 盘启动，进入 Live 模式——不安装也能先试用，确认 Wi-Fi、触板都正常再点安装。

安装时的三个关键点

  • Secure Boot：Ubuntu / Fedora 系直接支持，不用关；少数发行版需要进 BIOS 关掉它
  • 分区：双系统选「与 Windows 共存」让安装器自动分；手动分的话 / 给 50GB+，再留个 1GB 的 EFI 分区
  • 用户名：全小写、无空格无中文，未来的你会感谢现在的你

⚠️ 老手们爱说的 dd 命令写盘，写错盘符就是当场火化，新手请用图形工具。

第三步：装完后的第一件事

网络优化

换国内镜像源

默认软件源在国外，下载慢到怀疑人生。换成校园网内多半还免流量的国内镜像：

  • 清华 TUNA：mirrors.tuna.tsinghua.edu.cn
  • 中科大 USTC：mirrors.ustc.edu.cn
  • 阿里云：mirrors.aliyun.com

各镜像站首页都有「使用帮助」，搜你的发行版，复制粘贴即可。

日常体验

中文输入法与驱动

  • 输入法：fcitx5 + 中州韵（Rime）或自带拼音，开箱体验不输 Windows
  • NVIDIA 显卡：Ubuntu 系跑一句 sudo ubuntu-drivers autoinstall 完事；Arch 装 nvidia-open
  • 装 Timeshift 做个系统快照，折腾坏了十秒回滚——成年人的安全感
  • 注册 Flathub 仓库：图形软件一键装，省心

国内高校特供问题

校园网认证

网页认证（Portal）：恭喜，浏览器打开就完事，Linux 毫无障碍。

锐捷 / 深澜等客户端认证：有点麻烦但有解法——

  • 先加学校的 Linux 用户群 / 开源社团群，大概率有现成教程和打包好的客户端
  • 第三方开源客户端（如各类 minieap、drcom 实现）可解大部分学校
  • 终极方案：一台几十块的路由器刷 OpenWrt，让它替你认证，全宿舍设备跟着沾光

微信、QQ 能用吗？

能，而且这两年进步很大：

  • 微信：已有原生 Linux 版（NT 架构，官网提供 deb/rpm），日常聊天、文件传输都没问题，小程序等功能在逐步补齐
  • QQ：NT 架构新 QQ 官方支持 Linux，x86 和 ARM 都有，体验相当现代
  • 会议/办公：腾讯会议、飞书、钉钉均有官方 Linux 客户端；WPS 也有功能完整的 Linux 版

简单说：2026 年的今天，通讯办公已经不再是 Linux 的拦路虎了。
